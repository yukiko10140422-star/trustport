# CEO意思決定ログ: YURA ブランド深掘り調査

## 日付
2026-03-10

## 調査背景
前回の広範な先行事例調査を踏まえ、ブランドコンセプトが確定した「YURA」の方向性を前提に、5軸での的を絞った再調査を実施。

## 調査結果サマリー

### 軸1: 福島アパレル・三恵クレア
- 三恵クレアは南相馬市の婦人服縫製工場。50年近い実績、少量ロット対応（1〜10枚）、難素材のエキスパート
- 取引先: Paul Smith、Tomorrowland、SHIPS等の有名ブランド
- 福島県は繊維業者のOEM→ODM転換・ブランド化を推進中
- **EAST LOOP**が被災地発ファッションプロジェクトの最も成功した事例（累計3,500万円以上の還元）

### 軸2: 記憶×ファッション先行事例
- 「風化防止」を明確なミッションに据えたファッションブランドは**発見できなかった → ブルーオーシャン**
- Memory Lane LA（米）がコンセプト的に最も近い（記憶→未来の変換）
- 9/11 Memorial Museum Storeがメモリアル×アパレルの成功モデル
- 既存のチャリティーTは一過性が主流

### 軸3: 景観プリントアパレル
- 風景プリントT市場は成長中（CAGR 12.1%）
- 日本で「地域風景のアートブランド」は空白
- プリント技術: 初期はDTG/DTF → 量産はポリ素材なら昇華転写が最適
- 推奨価格帯: 5,000〜8,000円

### 軸4: ソーシャルインパクトD2C
- TOMSの1:1モデルは破綻 → 利益の1/3寄付モデルに転換
- Bombasは1:1モデルを効率的生産で持続
- ALL YOURSがCF→D2Cの日本の成功モデル（CAMPFIRE Award受賞）
- 推奨: 売上10〜15%寄付 + Makuakeでテストマーケティング

### 軸5: 競合・商標
- 「被災地×記憶×ファッション」の直接競合は**不在**
- 能登地震チャリティーTは多数あるが全て一過性
- NIWAKA「由良」がジュエリー分野で存在（アパレル区分とは異なる）
- 商標の早期確保が必要

## 総合分析: YURAの戦略的ポジション

### 強み（Strengths）
1. **カテゴリ創出の機会**: 「風化防止ファッション」という新領域の先行者になれる
2. **直接競合の不在**: ブルーオーシャン
3. **製造パートナーの存在**: 三恵クレア（高品質・少量対応）
4. **成長市場**: カスタムTシャツ/プリントT市場は拡大中

### リスク（Risks）
1. **「チャリティーTシャツ」と同一視されるリスク**: 差別化の明確な発信が必要
2. **商標リスク**: NIWAKA「由良」との混同可能性（低いが要確認）
3. **一過性で終わるリスク**: 継続的なコレクション展開の計画が必要
4. **市場教育コスト**: 新カテゴリの認知には時間がかかる

## CEO判断事項

### 即座に着手すべき事項（優先度: 高）
1. **商標出願**: 「YURA」「Yur-A」の第25類（被服）での商標調査・出願
2. **三恵クレアへのコンタクト**: Tシャツ/カットソー対応可否のヒアリング
3. **プリント技術のサンプル発注**: DTG/昇華転写でテスト印刷

### 短期的に進めるべき事項（1〜3ヶ月）
4. **Makuakeプロジェクト設計**: テストマーケティング計画（目標100〜300万円）
5. **写真家パートナーシップ**: 被災地風景の写真家リストアップ・打診
6. **寄付モデルの確定**: 売上10〜15%を復興支援に充当する仕組みの設計

### 中期的に検討すべき事項（3〜6ヶ月）
7. **D2C基盤構築**: Shopify等のEC基盤選定
8. **ブランドマニフェスト**: 「風化防止ファッション」のカテゴリ定義
9. **EAST LOOPとの連携**: 補完的な協力関係の模索
