# 事業コンセプト

## 「AIが全部やってくれるオールインワンDX」

<br>

| 項目 | 内容 |
|------|------|
| ターゲット | 地方小規模事業者（従業員5人以下） |
| プロダクト | AI搭載オールインワンDXプラットフォーム |
| 価格 | 月額 **¥980〜** で始められる |
| 差別化 | AI一人開発で大手SaaSの数倍のスピード |

> 既存SaaSは大規模チームで開発しているためスピードが遅い。
> AIを活用すれば**一人で開発から保守まで運用**でき、圧倒的なスピードで機能を提供できる。

---

# 6つの業務機能を1ツールに統合

<br>

| 機能 | できること |
|------|----------|
| **経理** | レシート撮影→自動仕訳、請求書自動作成 |
| **在庫** | 在庫数リアルタイム把握、自動発注アラート |
| **CRM** | 顧客情報一元管理、フォロー自動化 |
| **勤怠** | スマホ打刻、シフト管理、給与連携 |
| **予約** | オンライン予約受付、自動リマインド |
| **Web** | AIでLP自動生成、SNS連携 |

> **競合調査の結果: この6機能を単一で提供するツールは日本市場に存在しない**

---

# なぜこの事業か — 3つの確信

<br>

## I. AIの「現場適応」に価値がある

汎用SaaSの成長は鈍化する一方、AIを現場に適応させる役割は残り続ける

## II. 地方小規模事業者は取り残されている

DX導入率: 大企業 **66.0%** vs 中小企業 **40.6%** — 25.4ptの格差

## III. AI一人開発で戦える時代

AI搭載SaaSのARR到達速度: 従来の **3倍**（20ヶ月 vs 60ヶ月）
