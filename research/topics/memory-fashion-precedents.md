# 「記憶×ファッション」先行事例調査

## 調査日
2026-03-10

## 1. 災害の記憶をファッション/アートで表現するブランド

### 日本の事例

#### トリー・バーチ（Tory Burch）震災チャリティーTシャツ
- **概要**: 東日本大震災後、チャリティーTシャツを販売
- **特筆点**: 販売期間を約6カ月先の9月15日まで設定。短期ではなく長期的に支援し「記憶をつなぐ」意義を重視
- **示唆**: 一過性ではない「記憶の持続」をビジネスに組み込んだ先駆例
- **出典**: https://www.fashionsnap.com/article/fashion-force-miyata/

#### ファッション業界全体の震災対応
- アパレル業界は東日本大震災直後、物資支援（肌着、靴、ジャージ等）を実施
- デザイナーたちが「ファッションの力」で復興を支援する動きが広がった
- **出典**: https://www.fashionsnap.com/article/tokyodesigner-earthquake/

#### EAST LOOP
- 被災地の女性たちのニット製品プロジェクト（詳細は福島調査ファイル参照）
- 「記憶」そのものではなく「生活再建」にフォーカスした支援

### 海外の事例

#### 9/11 Memorial Museum Store
- **概要**: 9.11メモリアル博物館の公式ストアでアパレル・アクセサリーを販売
- **商品**: Tシャツ、キャップ、トートバッグ等のメモリアルグッズ
- **特徴**: 追悼と記憶の継承を目的とした「場」に紐づいたマーチャンダイズ
- **出典**: https://store.911memorial.org/collections/apparel

#### Memory Lane LA
- **概要**: ロサンゼルス発のアパレルブランド
- **コンセプト**: 「経験からのストーリーをアパレルで伝え、過去を振り返ることで未来への動きを作る」
- **示唆**: YURAの「記憶を力に変える」と非常に近いコンセプト。直接的な競合候補
- **出典**: https://memorylane.la/

#### Memory Threads Earth-Safe Apparel
- **概要**: 地球にやさしい素材を使った「記憶」テーマのアパレル
- **出典**: https://www.earthsafeapparel.com/

#### Beautiful Disaster Clothing
- **概要**: 試練と勝利を経験した女性を称えるブランド
- **コンセプト**: 勇気を奮い立たせ、力を与え、困難を乗り越えた女性を祝福
- **示唆**: 「災害」を直接テーマにしているわけではないが、「困難を力に変える」というメッセージはYURAと共通
- **出典**: https://www.bdrocks.com/

## 2. メモリアル・追悼テーマのアパレル

### Etsy Remembrance Clothing
- 個人クリエイターによる追悼・記念アパレルが多数出品
- 個人的な喪失（家族の死等）を記念するTシャツ・アクセサリーが中心
- カスタムメイド・少量生産が主流
- **出典**: https://www.etsy.com/market/remembrance_clothing

### 追悼イベント関連アパレル
- マラソン・チャリティーウォーク等のイベントTシャツ
- 「Remember」「Never Forget」等のメッセージTシャツ
- 一過性のイベントグッズが多く、継続的ブランドとして確立した例は少ない

## 3. 風化防止×ファッションの社会活動

### 災害風化の定量的研究
- 一般社団法人Code for Notoが「災害の風化を定量的に測定する学術研究」を発表
- 風化防止には「定期的な啓発」「教育の機会提供」「デジタル技術を活用した記憶の呼び起こし」が不可欠
- **出典**: https://note.com/code4noto/n/n7783a361df50

### ファッション×風化防止の現状
- 「風化防止」を明確にミッションとして掲げたファッションブランドは、調査時点で**発見できなかった**
- 多くは一時的なチャリティー販売や物資支援にとどまる
- **これはYURAにとって大きな差別化ポイント**

## 4. 戦争・災害の記憶保存クリエイティブプロジェクト

### デジタルアーカイブ系
- 震災のデジタルアーカイブ（写真・映像・証言）は多数存在
- しかし「アーカイブ×ファッション」の融合事例は非常に限られる

### アート系プロジェクト
- 震災をテーマにしたアート展示・インスタレーションは多数
- ファッションとの明確な接点を持つプロジェクトは希少

## 結論

「記憶×ファッション」の領域は以下の特徴がある:

1. **一過性が主流**: チャリティーTシャツ・イベントグッズが中心で、継続的なブランドとして確立した例は極めて少ない
2. **海外に近いコンセプト**: Memory Lane LAが「記憶→未来」の変換をアパレルで表現しており、YURAと最も近い
3. **風化防止ブランドは空白地帯**: 「風化防止」を明確なミッションに据えたファッションブランドは見つからなかった。**YURAが開拓できるブルーオーシャン**
4. **9/11モデルの参考**: メモリアル施設に紐づいたマーチャンダイズは成功事例

## YURAへの示唆

1. **「風化防止ファッション」というカテゴリ自体を創出する機会**: 既存の競合がほぼ不在。ブランディング上の大きなアドバンテージ
2. **一過性チャリティーとの差別化が必須**: YURAが「またチャリティーTシャツか」と思われないための工夫が重要
3. **Memory Lane LAの研究**: コンセプトが近いため、ポジショニングの差別化と学びの両面で分析すべき
4. **場所との紐づけ**: 9/11モデルのように、被災地のメモリアル施設・アーカイブとの連携が記憶の正統性を高める
5. **長期視点のビジネス設計**: トリー・バーチの6カ月販売の事例のように、「時間軸」をブランド設計に組み込むべき
