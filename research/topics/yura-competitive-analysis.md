2026-03-10

## 1. 震災テーマのアパレルブランド

### 東日本大震災関連

#### 継続的ブランドとして活動中
- **EAST LOOP / 東北クロッシェ村**: 被災地女性のニット製品プロジェクト。2011年〜継続。支援→自立→地域ビジネスに成長
- **ザ イノウエ ブラザーズ**: 東北の繊維工場と協力するソーシャルデザインスタジオ

#### 一時的なチャリティー活動

## 2. 復興支援チャリティーTシャツ・ブランドの特徴分析

### 共通パターン
1. **一過性**: ほぼすべてが災害直後〜1年程度の販売
2. **シンプルなメッセージ**: 「がんばろう○○」「SAVE ○○」等
3. **低価格帯**: 2,000〜5,000円
4. **寄付型**: 売上の一部または全額を寄付
5. **デザイン性が低い**: メッセージ重視でファッション性は二の次

### YURAとの決定的な違い
- 既存のチャリティーTはほぼすべて**一過性・イベント型**
- 「記憶の揺らぎを力に変える」という**哲学的コンセプト**を持つブランドは**皆無**
- 被災地の**景観の美しさ**をデザインに落とし込むアプローチは**独自**
- 「風化防止」を**ブランドミッション**として掲げる事例は**発見できず**

## 4. 競合ポジショニングマップ

### マッピング軸
- **横軸**: 一過性 ← → 継続的ブランド
- **縦軸**: チャリティー/支援型 ← → デザイン/アート型

```
デザイン/アート型
        │
        │   ★ YURA（目指すポジション）
        │
        │        Memory Lane LA
  ──────┼──────────────────────
一過性  │                    継続的
        │
        │  SHIPS ONE SHIPS PROJECT
チャリティー  THE ONENESS
/支援型  各種チャリティーT
        │
        │   EAST LOOP
```

### YURAの独自ポジション
YURAは以下の点で既存の競合と明確に差別化される:

1. **継続的ブランドとしての設計**: 一過性のチャリティーではない
2. **デザイン/アート性**: 被災地の景観を美しく表現するアート志向
3. **哲学的コンセプト**: 「記憶の揺らぎを力に変える」
4. **風化防止ミッション**: このポジションを占めるブランドは不在

### 最も近い競合
- **Memory Lane LA**（海外）: 「記憶→未来」の変換をアパレルで表現。ただし災害特化ではない
- **EAST LOOP**（国内）: 被災地発のファッションプロジェクト。ただしニット小物中心でアパレルブランドではない

## 結論

「被災地×記憶×ファッション」の領域で、YURAの直接的な競合は**存在しない**。既存のチャリティーTシャツは一過性で、デザイン性・ブランド性が低い。EAST LOOPは最も近い事例だが、アプローチが異なる（生計支援型 vs 記憶表現型）。YURAは「風化防止ファッション」という新カテゴリを創出できるポジションにある。

商標面では、NIWAKA「由良」がジュエリー分野で存在するが、アパレル区分では直接的な競合は確認されなかった。ただし正式な商標調査は必須。

## YURAへの示唆

1. **ブルーオーシャン**: 直接競合が不在であることは大きなアドバンテージ。先行者利益を取れる
2. **チャリティーTシャツとの明確な差別化**: デザイン性・継続性・哲学性で「チャリティーTシャツ」カテゴリから脱却すべき
3. **商標の早期確保**: 「YURA」「Yur-A」のアパレル区分（第25類）での商標出願を早急に進めるべき
4. **Memory Lane LAの研究**: コンセプトが最も近い海外ブランドとして、差別化ポイントを明確にすべき
5. **EAST LOOPとの連携可能性**: 競合ではなく補完的な関係。協力の余地あり
